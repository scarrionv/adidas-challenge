# Adidas Challenge

## Usage
 - Clone the repository
 - For **each** module in directory, to compile, test, build and create images, run this command:
```bash
mvn spring-boot:build-image
 ```
 - Execute docker compose to start
```bash
docker-compose up -d
 ```
